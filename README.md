# Pomodoro-Locker
Este projeto implementa um temporizador Pomodoro utilizando um Arduino, display LCD, micro servo, LEDs, botões e um buzzer. O sistema orienta o usuário por ciclos de foco e descanso, incluindo alertas sonoros e visuais, além de movimentação mecânica com o servo.

Funcionalidades do Projeto
  Exibe mensagens e contagem regressiva em um display LCD
  Utiliza LEDs para indicar foco (LED1) e descanso (LED2)
  Emite sinais sonoros via buzzer a cada transição
  Movimenta um servo motor conforme o estado do ciclo
  Permite iniciar o ciclo com um botão
  Possui botão de reset para interromper qualquer etapa
  Executa ciclos de foco, descanso curto e descanso longo

Ciclos do Pomodoro
O sistema segue a seguinte lógica:
  Foco: 1 minuto (25 min)
  Descanso curto: 1 minuto (5 min)
  Descanso longo: 1 minuto (15 min)
  Repetições do ciclo: 2 ciclos completos
**Importante: Para que o circuito pudesse ser avaliado, alterei o tempo para 1 min em cada etapa, alterar para os respectivos se utilizar o pomodoro.**
